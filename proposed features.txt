--Our proposed system is a desktop-based application that allows:

Storing patient details

Managing prescription information

Maintaining medicine records

Updating and deleting records easily

Viewing stored data quickly



--Additional Features
Planned Features of Our System:

Search patient records instantly

Track prescription history

Maintain medicine stock details

Billing record management

Date-wise transaction tracking

User-friendly interface for easy operation

Error messages for invalid data

Future plan: login authentication for security

These features will make the system more practical for real pharmacy use.

Technology Stack

We are using:

Python for backend logic

Tkinter for graphical interface

TTK Treeview to display data in table form

Datetime module for managing dates and timestamps
